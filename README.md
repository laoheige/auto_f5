# 🔒 Anti-Idle Click

> 防止网页因长时间无操作而过期的浏览器扩展

[![Chrome](https://img.shields.io/badge/Chrome-Compatible-blue)](https://www.google.com/chrome/)
[![Edge](https://img.shields.io/badge/Edge-Compatible-blue)](https://www.microsoft.com/edge/)
[![Manifest V3](https://img.shields.io/badge/Manifest-V3-green)](https://developer.chrome.com/docs/extensions/mv3/)

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| ⏱️ 自动点击 | 定时点击页面空白处，保持页面活跃 |
| 🎯 智能点击 | 只点击空白处，不会误触按钮和链接 |
| ⚙️ 灵活配置 | 自定义点击间隔（1-300秒） |
| 📊 运行统计 | 实时显示点击次数和运行时间 |
| 🌐 多标签支持 | 可选择当前标签或所有标签 |
| 💾 状态持久化 | 刷新页面后设置不丢失 |

## 📦 安装方法

### Chrome 浏览器

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 右上角打开「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目文件夹
5. 完成！点击扩展图标即可使用

### Edge 浏览器

1. 打开 Edge，地址栏输入 `edge://extensions/`
2. 左侧打开「开发人员模式」
3. 点击「加载解压缩」
4. 选择本项目文件夹
5. 完成！点击扩展图标即可使用

## 🚀 使用方法

```
1. 点击扩展图标打开控制面板
2. 设置点击间隔（默认10秒）
3. 选择点击范围（当前标签/所有标签）
4. 点击「启动」按钮
5. 放心离开，页面不会过期
```

## 🔧 工作原理

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   popup.js   │────▶│ background.js│────▶│  content.js  │
│   (控制面板)  │     │   (定时器)    │     │  (执行点击)   │
└──────────────┘     └──────────────┘     └──────────────┘
       │                    │                    │
       │                    ▼                    │
       │            ┌──────────────┐             │
       └───────────▶│chrome.storage│◀────────────┘
                    │  (数据存储)   │
                    └──────────────┘
```

1. **popup.js** - 用户界面，接收配置并发送指令
2. **background.js** - 后台服务，管理定时器和状态
3. **content.js** - 注入页面，执行实际的点击操作
4. **chrome.storage** - 持久化存储，保存设置和统计数据

## ⚠️ 注意事项

- 部分网站可能会检测自动化点击行为
- 建议在需要保持登录的内部系统使用
- 不要在需要真实用户交互的场景使用
